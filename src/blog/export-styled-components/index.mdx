I have been a big fan of styled-components for a while now. It's been my go to styling library for all my most recent projects. Over the years I have adopted various patterns for developing components and I have been using this pattern for quite a while now. Let's get to it.

Most of the time in styled-components when we create a component we are going to use the `styled` function. This returns a react component that can then be rendered inside of other react components.
